Tic Tac Toe

Purpose and Audience (2-3 sentences)

My audience will be users who enjoy a simple Tic Tac Toe game. They can learn and understand the game through this website.

Description of the Dynamic Elements

JavaScript will allow me to use buttons and arrays to display the game, create a turn system, and calculate the winning conditions of the board.

Pseudocode - Computational Thinking

I’ll create an array for the board’s squares, create a turn system that flips each turn, a win checker, a series of buttons to create the tic tac toe game. Some of that may change depending on how things actually turn out, but this is the basis of my pseudo code plan.
